# Movie-App

This is a project of Movie App using HTML,SCSS

For stepup the project on your system use:

<h4>Step 1</h4>

`git clone https://github.com/Abhishek9608/Movie-App.git`

<h4>Step 2</h4>

`npm install`

<h4>Step 3</h4>

`npm run compile:sass`

<h4>Step 4</h4>

check your server at localhost:8000
